## Solution
``` java
public int findNumbers(int[] nums) {
	int even = 0;
	for(int i = 0; i < nums.length; i++) {
		if(((int)Math.log10((double)nums[i])+1)%2 == 0 )
			even++;
	}
	return even;
}
```
