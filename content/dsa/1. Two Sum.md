## Compliment Method
``` java
int getPairsCount(int[] arr, int n, int k) {
	HashMap<Integer, Integer> map = new HashMap<>();
	int count = 0;
	for(int num : arr) {
		Integer val = map.get(k-num);
		if(val != null) count+= val;
		map.put(num, map.getOrDefault(num, 0)+1);
	}
	return count;
}
```
