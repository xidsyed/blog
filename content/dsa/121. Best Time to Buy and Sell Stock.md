## Solution
``` java
public int maxProfit (int[] prices){
	int max_price = Integer.MIN_VALUE, maxProfit = 0;
	for (int i = prices.length-1; i >= 0 ; i--) {
		max_price = Math.max(max_price, prices[i]);
		maxProfit = Math.max(maxProfit , max_price - prices[i]);
	}
	return maxProfit;
}
```
